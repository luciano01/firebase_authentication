## For developers
> To clone locally
```
$ git clone git@github.com:luciano01/firebase_authentication.git
$ cd firebase_authentication
```
> Open the project in an editor, if you use VS Code, follow this
```
$ cd firebase_authentication
$ code .
```

> To run, follow this on terminal
```
$ flutter clean
$ flutter pub get
$ flutter run --release
```
